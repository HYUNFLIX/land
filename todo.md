# 동래 푸르지오 에듀포레 웹페이지 개선 아이디어 (To-Do List)

현재 프리미엄 브랜드 아이덴티티와 구조는 잘 갖춰져 있습니다. 향후 마케팅 효율 및 전환율을 한 단계 더 끌어올리기 위한 추가 개선 방안을 제안합니다.

## 1. 퍼포먼스 마케팅 및 데이터 분석 연동
- [ ] **Google Analytics 4 (GA4) 연동**: 방문자의 체류 시간, 스크롤 뎁스, 이탈 구간 세밀 분석
- [ ] **Meta(Facebook) Pixel & Kakao Pixel 설치**: 광고를 통한 유입자 추적 및 리타겟팅(리마케팅) 모수 확보
- [ ] **전환(Conversion) 이벤트 세팅**: '전화번호 클릭', '카카오톡 버튼 클릭', '관심고객 등록 완료' 행동을 핵심 전환 값으로 설정하여 광고 효율 극대화

## 2. 사용자 경험(UX) 고도화
- [ ] **카카오 싱크 (Kakao Sync) 1초 간편 등록 지원**: 고객이 일일이 이름/전화번호를 치지 않고 '카카오로 1초 만에 관심고객 등록하기' 버튼을 도입해 등록 전환율 대폭 상승 유도
- [ ] **3D 브이투어 (VR Tour) 연동**: 평면정보 섹션에 'VR 모델하우스 둘러보기' 버튼을 추가하여 고객 체류 시간을 극대화하고 오프라인 방문 유도
- [ ] **현장 라이브 웹캠 또는 드론 영상 배경화면화**: 첫 화면(Hero) 조감도를 정적인 이미지가 아닌 드론이 촬영한 현장/주변(온천천 등) 4K 짧은 영상 루프로 변경하여 생동감 추가
- [ ] **지도 API 연동 고도화**: 단순 구글 맵 대신 '네이버 지도 API'나 '카카오맵 API'를 사용하여 주변 핵심 학군(충렬초, 충렬중)이나 전철역(안락역)까지의 도보 이동 시간/거리 시각화 

## 3. 웹사이트 로딩 속도 및 SEO 최적화
- [ ] **이미지 파일 경량화 (WebP 형식 전환)**: 투시도 등 대용량 이미지를 `.webp` 확장자로 변환하여 모바일 환경에서도 즉각적으로 화면이 뜨게 퍼포먼스 최적화 (`hero_apartment.png` 등 교체 권장)
- [ ] **SEO (검색엔진 최적화) 메타 태그 강화**: 네이버, 구글 검색 시 상단 노출을 위해 `Open Graph` (OG 이미지), `Twitter Card`, 그리고 구성을 설명하는 `Schema.org` (JSON-LD) 마크업 삽입
- [ ] **브랜드 스플래시(Loading) 애니메이션 추가**: 처음 접속 시 1~2초 동안 푸르지오 로고가 은은하게 나타나는 프리미엄 로딩 효과 적용

## 4. 운영 및 관리 편의성
- [ ] **동적 공지사항 팝업 모듈(Cookie-base)**: "오늘 하루 보지 않기" 기능이 탑재된 이벤트/공지사항 모달창 모듈 구현 (청약 일정 등 주요 이벤트 알림용)
- [ ] **구글 시트 연동(Google Sheets API)**: 관심 고객 등록 시, 서버를 구축하지 않고도 마케팅 팀이 즉각 사용할 수 있도록 데이터가 구글 스프레드시트에 실시간으로 기록되게 설정
